
Canonical [xarray](https://xarray.dev/) integration for [unxt](https://github.com/GalacticDynamics/unxt).

`unxts.xarray` is the canonical location for xarray integration, superseding the deprecated `unxt-xarray` package.

Install: `pip install unxts.xarray`
