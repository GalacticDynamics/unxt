# `unxts.hypothesis`

```{toctree}
:maxdepth: 1
:hidden:
```

Canonical [Hypothesis](https://hypothesis.readthedocs.io/) strategies for [unxt](https://github.com/GalacticDynamics/unxt).

`unxts.hypothesis` is the canonical location for the Hypothesis strategies previously provided by `unxt-hypothesis`. The legacy `unxt-hypothesis` package remains available as a backward-compatible shim.

Install: `pip install unxts.hypothesis`
