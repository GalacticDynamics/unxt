# Sample Data Files

This directory contains sample NetCDF files used in the documentation examples.

## Files

- `sample_data.nc`: Example atmospheric measurements dataset with CF-compliant
  unit metadata
  - Generated by `generate_sample_data.py`
  - Contains temperature, pressure, and distance measurements
  - Used in the "Loading from NetCDF" example in `xarray-guide.md`

## Regenerating the Data

To regenerate the sample data file:

```bash
uv run python generate_sample_data.py
```

This is useful if you need to update the sample data structure or values.
