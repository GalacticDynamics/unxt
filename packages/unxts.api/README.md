# unxts.api

Abstract dispatch API definitions for [unxt](https://github.com/GalacticDynamics/unxt).

This is the canonical package (`unxts.api`). It defines the abstract single-dispatch functions that libraries implement to support `unxt`-style unit-aware quantities.

## Install

```bash
pip install unxts.api
```

## Usage

```python
import unxts.api as uapi
```
