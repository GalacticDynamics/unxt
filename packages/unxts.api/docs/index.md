```{toctree}
:maxdepth: 1
:hidden:
```

Canonical abstract dispatch API for [unxt](https://github.com/GalacticDynamics/unxt).

`unxts.api` is the canonical location for the abstract dispatch interfaces previously provided by `unxt-api`. The legacy `unxt-api` package remains available as a backward-compatible shim.

Install: `pip install unxts.api`
