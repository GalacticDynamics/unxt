# unxts.xarray

xarray integration for [unxt](https://github.com/GalacticDynamics/unxt).

This is the canonical package (`unxts.xarray`). It provides xarray accessors and conversion utilities for unxt's JAX-based quantities.

## Install

```bash
pip install unxts.xarray
```

## Usage

```python
import unxts.xarray
```
